# Is x a SharePoint URL?

Is x a SharePoint URL?

## Usage

``` r
is_sp_url(x)

is_sp_site_url(x)

is_sp_drive_url(x)

is_sp_site_page_url(x)

is_sp_doc_url(x)

is_sp_type_url(x, type = "w")

is_sp_folder_url(x)
```

## Arguments

- x:

  Object to test if it is a URL.

- type:

  Type of URL to test against.
